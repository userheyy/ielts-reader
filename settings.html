设置

DeepSeek AI

填好 API Key(接口密钥,相当于你调用 AI 的账号凭证)后,站内的 AI 功能才能用。 Key 在 DeepSeek 开放平台(platform.deepseek.com)注册后创建,按用量计费。

安全提示:Key 只保存在本浏览器的 localStorage(浏览器本地存储)里,本站不会把它上传到任何服务器。 在公共电脑上用完请点「清除 Key」;如果 Key 曾在聊天记录、截图里暴露过,请立刻到 DeepSeek 后台重置。

朗读设置

英语发音人 + 语速统一在这里设置,全站朗读(阅读句子、词库单词、复习卡)都用你选的音色和语速。 用的是浏览器自带的 speechSynthesis,不联网、不上传。

找不到英语音色?到系统「语音 / 语言与地区」里添加英语语言包后刷新本页。

数据

生词、导入的文章、朗读设置等数据都存在本浏览器里,换电脑或清浏览器缓存会丢。 备份方法:到 阅读 页面,点右上角的「全量备份」导出一份 JSON 文件保存好; 在新电脑上用旁边的「恢复全量备份」把这份文件导回来即可。
